XML and JavaScript

By Asa Morris

What is XML?

XML (eXtensible Markup Language) is a markup language that's used to store data. Information is stored within custom-made tags, similar to HTML tags, which provide context for the data.

How is XML used?

XML is used to store and share data in plain text. This is a preferrable method for storing data because it has no dependencies that can vary between operating systems or hardware.

What does XML look like?

XML looks similar to HTML, with custom-nade tags and a cascading structure of tags within tags.

<student>
<name>John</name>
<id>0296835</id>
<class>Senior</class>
<grade>B-</grade>
</student>

What is JavaScript?

JavaScript (JS) is an object-oriented scripting language that's used to manipulate web pages. JS can do a number of things, including dynamically updating content, creating graphics, and adding interactivity to web elements. JavaScript is interpreted through client-side processing, meaning it's the browser's responsibility to run the JS code retrieved from the web server.

How is Javascript related to HTML and CSS?

HTML, CSS, and JS files are all sent to a browser together. HTML creates the structure of a website using elements. CSS stylizes the HTML elements and their properties to make the website visually appealing. JS dynamically modifies both the HTML and CSS files based on specific conditions, such as buttons being clicked or a page being loaded.

What are some common JavaScript elements, properties, methods, and functions?

  • Elements: The browser window itself, the entire web page, individual elements within the page.
  • Properties: .color, .height, .width
  • Methods: .write(), .submit(), onclick()
  • Functions: .toString(), .length(), .substring()
