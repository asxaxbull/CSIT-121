What Are Forms?

Forms are an HTML element that allow a website's users to enter information that is then collected and used by the website's servers. Forms are made up of form elements — pieces of HTML code that represent fields, buttons, and more — which are then contained in a <form> tag.

The Input Tag

The <input> tag is a multifaceted HTML element that allows a website's users to input data in a number of different ways. There are a number of different types of inputs, including text boxes, radio buttons, checkboxes, and more.

The Text Area Tag

The <textarea> tag is a large text box that allows users to enter large blocks of text. This tag is useful for entering comments or submitting answers

The Select Tag

The <select> tag allows the user to select an option from a dropdown. This is useful if there are too many options to display in a radio button.


The Button Tag

The <button> tag allows the user creates an interactable area that the user can click on to perform an action. This tag can be used to create submit buttons, reset buttons, and more.



The Fieldset Tag

The <fieldset> tag creates a stylized region into which form elements can be placed. Fieldsets are used to group form elements together and label the form itself in a way that's visible to users.
